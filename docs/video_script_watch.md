# KpForU 手表端 - 视频介绍讲述稿

**时长建议：2.5-3分钟**

---

## 开场 (15秒)

大家好，我是双端APP，KpForU手表端的演示。当你在学习中需要**极速专注和轻量提醒**时，手表就是你的最佳伴侣。

手表端聚焦三大核心场景：专注计时、复习提醒、快速语音卡片创建。所有功能都为小屏幕优化，交互极简，续航友好。

---

## 第一部分：核心功能展示 (90秒)

### 1. 主屏导航 (10秒)
**[展示手表首屏]**

手表端入口非常简洁，三个主要功能区：
- **专注计时**（左上）：进入时间流专注模式
- **记忆卡片**（中央大按钮）：今日待复习卡片中心
- **菜单**（右上）：模板、历史记录、设置

整个UI遵循极简原则：**最少点击次数，最直接的信息**。

### 2. 专注计时 - 时间流核心 (40秒)
**[演示专注页面]**

这是手表端最重要的功能。

**[展示当前阶段]**
- 屏幕中央显示当前阶段名（如"深度学习"、"休息"）
- 大号倒计时显示剩余时间（mm:ss格式）
- 下方进度环实时反映当前阶段进度
- 右上显示"循环进度"（第3/5个循环）

**[演示控制按钮]**

底部三个按钮：
- **开始/暂停**：控制计时
- **下一个**：跳过当前阶段，立即进入下一阶段
- **结束**：整个时间流会话结束

**[演示通知与振动]**

每个阶段切换时：
- 📳 手表振动提醒
- 🔔 全屏弹窗提示（"学习开始"或"休息时间"）
- 音量友好，不干扰他人

可选功能：如果你设定了"定时振动"，在学习阶段会每隔5分钟振动一次，帮助容易分心的人保持警惕。

**[讲述技术细节]**

所有这一切都由**前台服务（Foreground Service）**支撑，确保：
- 熄屏或切后台后计时**依然精确进行**
- 震动和提醒**100%可靠触发**
- 系统内存压力下**稳定不卡顿**

### 3. 结束会话确认流程 (20秒)
**[演示结束按钮 → 确认屏幕]**

当专注结束（无论是自然完成还是提前结束），应用会：

**[展示确认卡片]**
- 弹出一个轻量化的确认屏幕
- 显示本次会话的统计：实际专注时长、是否提前结束、当前阶段
- 倒计时5秒自动保存，或你也可以手动确认

这个设计考虑了：
- 有人喜欢"手动确认"的仪式感
- 有人希望"自动保存"的便捷性
- **可在手表端设置偏好，手机端同步**

会话数据上传到云端后，手机端可以看到完整的专注历史。

### 4. 今日复习提醒 (20秒)
**[展示Cards页面]**

点击中央的"记忆卡片"按钮，进入复习模式。

**[展示卡片列表]**
屏幕顶部显示关键指标：
- 📋 **今日待复习：8张**
- ✅ **已完成：3张**
- ⏰ **下一复习：14:30**

**[演示卡片翻转]**

列表中每张卡片支持：
- **点击翻转**：看问题 ↔ 看答案
- 卡片标签显示所属分类
- 卡片背景色区分复习盒数（视觉反馈）

**[演示操作按钮]**

每张卡片下方始终有两个按钮：
- ✅ **完成**：这道题我掌握了，升级到下一个复习周期
- ⏱️ **稍后**：这次先不复习，延后1/2/3天

**操作即刻同步到云端，手机端同步更新。**

### 5. 语音快速卡片创建 (20秒)
**[展示语音录入界面]**

**这是手表的独特优势——快速捕捉想法。**

**[演示录音过程]**

在任何时刻，点击菜单中的"语音创建"：
- 点击大红按钮开始录音
- 说出你要记住的内容（最长30秒，≤500KB）
- 录音完成，点击上传

**[展示上传状态]**
- 状态：处理中 🔄
- 应用调用云端AI模型（通义千问）进行转写
- 转写完成后显示识别的文本
- 自动生成卡片草稿

**核心好处：**
- 💡 想到就说，不必停下来打字
- ⚡ 边走边学，碎片化场景完美适应
- 🔗 草稿同步到手机端，可进一步精修

## 第二部分：现有高级特性 (30秒)

### 6. 离线可用性 (15秒)
**[讲述使用场景]**

未绑定云端的情况下，手表依然可以：
- ✓ 正常使用专注计时（本地保存）
- ✓ 查看之前同步的卡片数据
- ✓ 完成/延后卡片操作（本地记录）

一旦网络恢复或重新绑定，所有本地操作会自动同步。

**这是为出地铁、进地下室、海外出差等场景设计的。**

### 7. 模板管理 (15秒)
**[展示模板列表]**

从菜单进入"模板"：
- 查看所有时间流模板
- 删除不需要的模板
- 新增模板（可从手机端新增后同步）

每次启动专注前，你可以快速选择一个模板。

## 手表端独特的讲述重点

1. **极简哲学** - 不是简陋，而是经过深思熟虑的最小化设计
2. **可靠性** - 前台服务、续航优化是核心竞争力
3. **碎片化学习** - 手表的天然优势：随处可得、随时可用
4. **双端协同** - 手表是补充，不是替代；两端各司其职
5. **健康学习** - 提醒频率、定时振动等都考虑了学习节奏和身心健康
