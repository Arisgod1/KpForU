# KpForU 手机端 - 视频介绍讲述稿

**时长建议：3-4分钟**

---

## 开场 (15秒)

大家好，我是双端APP，KpForU的演示。KpForU是一款帮助你保持专注、坚持复习的智能学习应用。手机端作为核心交互中心，提供卡片管理、学习数据总览和AI驱动的学习建议。

让我为大家展示一下它能做什么。

---

## 第一部分：核心功能展示 (90秒)

### 1. 首屏概览 (20秒)
**[展示Dashboard屏幕]**

打开应用，首屏显示四个关键信息：
- **今日待复习数**：一眼看到今天有多少张卡片需要复习
- **模板数量**：管理的时间流模板总数
- **本周专注统计**：你投入的学习时间总量
- **AI学习总结**：今天的学习效果评价和改进建议

这个设计的核心理念是：**信息密度高，但一眼清晰**。

### 2. 时间流模板管理 (25秒)
**[展示TimeFlow Templates屏幕]**

**"时间流"** 是番茄工作法的升级版。你可以自定义多个模板：
- **25/5 模式**：25分钟学习 + 5分钟休息（适合短期冲刺）
- **45/10 模式**：45分钟学习 + 10分钟休息（适合深度学习）
- **60/10 模式**：60分钟学习 + 10分钟休息（适合专项突破）

**[演示新增按钮]**

点击"+"可以新增自定义模板。每个模板支持：
- 多阶段循环（可设定循环次数或截止时间）
- 自定义阶段名称（"深度学习"、"思考休息"等）
- 可选的定时振动提醒

这些模板会实时同步到手表，让你开始专注时无需重复配置。

### 3. 复习待办中心 (30秒)
**[展示Reviews页面]**

这是手机端最重要的功能——**复习待办中心**。

**[演示列表]**

应用会根据Leitner间隔系统自动计算你今天应该复习哪些卡片：
- 卡片采用分级复习：第1~5盒，间隔从1天递增到14天
- 卡片正反面可点击翻转查看
- 每张卡片都显示当前复习盒数和下次复习时间

**[演示操作]**

你可以对每张卡片进行操作：
- **"完成"**：这张卡片掌握了，升级到下一个复习周期
- **"稍后"**：今天先不复习，延后1/2/3天再说
- **"删除"**：这张卡片不需要了

所有操作实时同步到云端，手表也能立即看到最新状态。

### 4. 卡片管理 (15秒)
**[展示Cards列表和编辑屏幕]**

你可以手动创建和编辑卡片：
- 输入问题（正面）和答案（背面）
- 添加标签分类
- 编辑和删除卡片

**[展示新增按钮]**

或者，你可以从手表端快速录音创建卡片，我们稍后会讲到。

## 第二部分：现有高级功能 (45秒)

### 5. 语音草稿上传 (25秒)
**[展示Voice Drafts页面]**

手表端录制的语音卡片会在这里显示为"草稿"：

**[演示上传流程]**
- 选择本地音频文件
- 点击上传
- 应用调用云端AI模型（通义千问）进行语音转写
- 转写完成后显示识别结果

你可以：
- 预览转写文本
- 编辑纠正识别错误
- 保存成正式卡片
- 删除不需要的草稿

这个流程大大加速了"想到就记录"的工作流。

### 6. AI学习总结与建议 (20秒)
**[展示Summary页面 - 目前是占位]**

**[讲述功能愿景]**

基于你的专注记录和复习数据，AI会生成：

**今日总结：**
- 你今天投入了多少专注时间
- 完成了几次复习
- 学习节奏是否稳定

**AI建议：**
- 如果你频繁提前结束专注，AI会建议缩短学习段时长
- 如果你经常延后复习，AI会建议降低每天的复习量
- 基于你的学习数据，给出更合理的时间流配置

这些建议是**可落地的、有针对性的**，不是泛泛而谈。

**[展示想象效果或UI原型]**

这是我们最令人兴奋的功能——**AI生成学习PPT**。

**工作流程：**

4. 自动生成结构化的PPT演示：

**应用场景：**
- 准备小组分享前快速整理思路

### 9. 双端换壁纸功能 (20秒)
**[展示壁纸库效果或设计稿]**
